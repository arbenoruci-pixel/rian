What changed:
- Adds BASE_CACHE/T-code stale transport detection before showing Pastrimi problem cards.

  - id

  - client_tcode
  - data->>id
  - data->client->>code
  - data->client->>tcode

- BASE_CACHE rows are shown as Pastrimi problems only when no matching DB transport order is found and there is no local/outbox detail match.
