PATCH: RUNTIME CORE V1 — AppRoot safety modules removed from lazy chunks

Scope:
- No DB changes
- No write flow changes
- No outbox/offline mutation format changes

1. src/AppRoot.jsx
- ChunkLoadRuntime, RootResumeWatchdog, and ServiceWorkerRegister are now static imports in AppRoot.

- Added optional repairHref / repairLabel action.
- Runtime module fallback can now show RIPARO APP linking to /pwa-repair.html?from=runtime_import_failure.
