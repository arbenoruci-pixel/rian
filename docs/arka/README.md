# ARKA — MODULAR (MENU → MODULE)

Qëllimi: kur preket **ARKA** nga Home, hapet një **MENY e pastër** me module.

## Rrugët
- `/arka` → Menu
- `/arka/buxheti` → CASH + buxheti i kompanisë + muaji (ndarja)
- `/arka/puntoret` → Lista puntorëve (add/edit/remove) + role (ADMIN/JO)
- `/arka/borqet` → Borxhe (na kanë borxh / i kemi borxh)
- `/arka/investimet` → Owners + investime + % split
- `/arka/reset` → Reset (endpoint server: `/api/admin/reset`)

## Rregulla
- Mos e përziej ARKËN me Transportin.
- Çdo modul e ka faqen e vet → ndryshimet bëhen vetëm në atë folder.
- UI: ALL CAPS, dark, high contrast.
