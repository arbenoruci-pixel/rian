
1) GLOBAL SYNC BOOT

Import SyncBoot in app/layout.js

import SyncBoot from '@/components/SyncBoot'

Then place it inside <body>:

<body>
  <SyncBoot />
  {children}
</body>


2) DOCTOR PAGE MONITOR

Import component in:

app/doctor/page.jsx

import OutboxMonitor from '@/components/OutboxMonitor'

Then render somewhere in the page:

<OutboxMonitor />


3) RESULT

You will see:

OUTBOX MONITOR
Pending | Failed | Total

and a FORCE SYNC button.

This reads localStorage key:

tepiha_outbox_v1

which is created by syncManager.js.
